# cmu-ece-thesis-template
CMU ECE PhD Thesis LaTeX template

A LaTeX template for PhD dissertations that conforms to the ECE
department's requirements.

This is a template for a CMU ECE dissertation.  It conforms to the
requirements from ECE as of May 2020. See [CMU College of Engineering
Thesis and dissertation
standards](https://engineering.cmu.edu/education/academic-policies/graduate-policies/thesis-dissertation.html).

It is put together from 2 sources:
1. [Raja Sambasivan repository](https://github.com/RS1999ent/cmu_ece_dissertation_template)
2. cmuthesis2.sty, version 2.0ame, February 2003 by Dinesh Das obtained from Doru Thom Popovici.
